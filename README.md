# MCC Engineering Website

A static website for MCC Engineering, built with HTML, CSS, and JavaScript.

## Project Structure

```
├── index.html       # Main page
├── css/
│   └── styles.css   # Stylesheet
├── js/
│   └── main.js      # JavaScript (smooth scroll, contact form)
└── .gitignore
```

## Getting Started

Open `index.html` directly in a browser, or serve the project with any static file server:

```bash
npx serve .
```

## Sections

- **Hero** – Welcome banner with a call-to-action
- **About** – Overview of MCC Engineering
- **Services** – Consulting, Design, and Project Management cards
- **Contact** – Simple contact form
